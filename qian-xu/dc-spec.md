# 龙胶囊档案

## DC规格

每个龙胶囊都是独一无二的，并由381种可能的特征属性以编程方式生成，包括表情、魔法角、服装、装备等。 生成的龙胶囊有上百万种，发行版本仅有随机的10000枚，包含一枚原始龙胶囊彩蛋，所有的龙胶囊都是孤品，但有些龙胶囊比其他的更稀有。
